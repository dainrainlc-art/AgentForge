## 📝 技术笔记

### LinkedIn API 集成经验

1. **OAuth 2.0 流程**
   - 使用 `state` 参数防止 CSRF
   - 令牌有效期 1 小时，需要刷新机制
   - 重定向 URI 必须匹配

2. **API 限制**
   - 不支持直接发送连接请求
   - 消息 API 需要特殊权限
   - Jobs API 需要公司页面权限

3. **最佳实践**
   - 实现重试机制（使用 tenacity）
   - 缓存个人资料数据
   - 定期刷新令牌

### 自进化系统经验

1. **定时任务**
   - 使用 asyncio 调度
   - 错误处理很重要
   - 日志记录详细

2. **AI 分析**
   - LLM 响应需要容错处理
   - JSON 解析可能失败
   - 设置合理的超时时间
