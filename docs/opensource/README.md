# AgentForge 开源资源文档索引

## 核心框架

### OpenAkita (开源自进化AI助手框架)
- **GitHub**: https://github.com/openakita/openakita
- **文档**: https://docs.openakita.ai
- **用途**: AgentForge核心AI助手框架
- **关键特性**:
  - 自进化能力（每日记忆巩固与自我检查）
  - 多LLM动态路由
  - Agent Skills和MCP开放标准
  - 长期记忆系统

### n8n (开源工作流自动化平台)
- **GitHub**: https://github.com/n8n-io/n8n
- **文档**: https://docs.n8n.io
- **用途**: 工作流编排引擎
- **关键特性**:
  - 200+原生集成节点
  - 可视化工作流编辑
  - 自托管部署
  - Webhook支持

## 知识管理

### Obsidian
- **官网**: https://obsidian.md
- **插件市场**: https://obsidian.md/plugins
- **用途**: 本地知识库管理
- **关键插件**:
  - obsidian-git: 自动Git同步
  - obsidian-shell-commands: 命令行增强
  - templater: 模板系统

### Notion
- **API文档**: https://developers.notion.com
- **用途**: 云端协作空间
- **关键功能**:
  - 数据库API
  - 页面操作API
  - 搜索API
  - Webhook集成
