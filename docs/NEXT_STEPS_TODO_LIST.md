## 🎯 立即执行计划

### 第一步：前端构建（现在开始）

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 构建生产版本
npm run build

# 4. 测试构建结果
npm run preview
```

### 第二步：启动监控（前端构建完成后）

```bash
# 1. 启动监控栈
docker-compose -f docker-compose.monitoring.yml up -d

# 2. 查看状态
docker-compose -f docker-compose.monitoring.yml ps

# 3. 访问 Grafana
# http://localhost:3000 (admin/admin)
```

### 第三步：配置 HTTPS（监控启动后）

```bash
# 1. 更新 SSL 证书
./scripts/update-ssl-cert.sh

# 2. 重启 Nginx
docker-compose -f docker-compose.prod.yml restart agentforge-nginx

# 3. 测试 HTTPS
curl -k https://localhost
```
